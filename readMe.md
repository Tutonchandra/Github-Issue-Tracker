
  # What is the difference between var, let, and const?

1️⃣ Answer:

  All three var, let, and const are used to declare variables in JavaScript, but they behave differently.

  * var is function-Scoped and it can be redeclared in JavaScript. Sometimes it can be accessed both inside and outside of a block.

  * let is block-scoped and cannot be redeclared, but it can be reassigned.

  * const is also block-scoped. It cannot be redeclared or reassigned. However, if it is an object or array, its internal values can be  changed.

  # What is the difference between map(), filter(), and forEach()?
3️⃣ Answer:

  map(), filter(), and forEach() are JavaScript array methods used to loop through elements. 

  * The map() method creates a new array by changing each element. 

  * The filter() method creates a new array with elements that match a condition. 

  * The forEach() method only loops through the array and performs an action, but it does not return a new array.
